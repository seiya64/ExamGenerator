# Task 01: Scaffold Angular App

## Goal

Create the initial Angular v21 stable application structure with TypeScript, routing, SCSS, and a working default page.

## Implementation

- Use Angular CLI to create the app in the repository root.
- Configure:
  - Standalone Angular app.
  - Routing enabled.
  - SCSS styling.
  - Strict TypeScript settings.
- Keep the first screen simple, such as a temporary app title.
- Do not add exam behavior yet.

## Acceptance Checks

- `npm install` completes.
- `npm start` or `ng serve` starts the app.
- The browser shows a non-empty Angular page.
- `npm run build` passes.
